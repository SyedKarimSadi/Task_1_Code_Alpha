# Task_1_Code_Alpha
# 🎓 GPA / CGPA Calculator (C++)

## 📌 Overview

A simple console-based C++ program to calculate **Semester GPA** and **Overall CGPA** using user input.

---

## ✨ Features

* Calculate GPA
* Calculate CGPA
* Menu-driven program
* Uses credit hours & grade points

---

## ▶️ How to Run

```bash
g++ "Task 1.cpp" -o gpa
./gpa
```

---

## 🧮 Formula

**GPA =** Σ(GradePoint × CreditHours) / Σ(CreditHours)
